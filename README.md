# SpaceX Falcon 9 Landing Prediction

This project analyzes SpaceX Falcon 9 launch data to predict whether the first stage will land successfully.

## Project Steps
1. Data collection using SpaceX API
2. Web scraping Falcon 9 launch data from Wikipedia
3. Data wrangling and cleaning
4. Exploratory data analysis with visualization
5. SQL analysis
6. Interactive map with Folium
7. Dashboard with Plotly Dash
8. Machine learning classification models

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- BeautifulSoup
- Folium
- Plotly Dash
- Scikit-learn
- SQL

## Models Used
- Logistic Regression
- Support Vector Machine
- Decision Tree
- K-Nearest Neighbors

## Goal
Predict Falcon 9 first-stage landing success and identify important factors affecting launch outcomes.
